# Quantitative Study of Conditions of Civil War Recovery Based on International Intervention:
## Introduction
This is the hub for my work on researching the requirements for success of civil war recovery in different kinds of civil wars -- specifically, those that feature a heavy amount of external involvement. Inspired by the ongoing Sudanese civil war and the tentative conclusion to the Syrian civil war and [Sharif's "Predicting the End of the Syrian Conflict" (2019)](https://doi.org/10.1080/1057610X.2018.1538153), this studies whether internationalized and non-internationalized civil wars have different requirements for reconstruction. For the full study, see the [final report](./DATS4001_report.pdf)

## Reproduction Notes: 
Almost all data used is made available here -- the one exception is the V-Dem dataset, which had too large of a file size (and also my source for GDP per capita, whose data I sourced in V-Dem). However, the first ten rows of that dataset are found in the `sampledata` subdirectory. One dataset that was sourced through an R package was oil rents from the World Bank. 

To reproduce the results from the cleaned datasets, run the code in `code/qca.Rmd`.
