\section*{Weekly Progress (5 points)}
% List below at least FIVE significant updates
\begin{enumerate}
	\item Committing to QCA as my research method -- given the number of cases it looks like I'll have per category (uncomitted, since I'm still tailoring the start year of my study), it seems like a very reasonable research method that provides interesting results. The primary textbook I'm using for this is Patrick A. Mello's \emph{Qualitative Comparative Analysis: An Introduction to Research Design and Application}, though I've consulted some other resources (this is just a good comprehensive introduction). I think I'll find some example versions for political science to learn from past cases. It also lets you have some pretty unique insights!
    \item Made the switch to Correlates of War as my conflict dataset -- having some supplemental information from UCDP may end up being useful, so I'll keep my pruned version of that dataset in case it becomes useful (and the code for deriving it). It comes down to what information I wish I had on my overall dataset and actually getting there. I think I'm going to stick with R up until the actual analysis as well.
    \item Performed some EDA on the cases I have (particularly comparing between the internationalized and non-internationalized cases). I may end up having issues with the classification that COW has done already for that, but it seems to be a reasonable enough? I'm still going to do some digging on this if I can. Quoth the 4.0 codebook, "[i]ntra-state wars are classified as 'internationalized' when an outside state or states intervenes in the war. The war remains classified as “internationalized” as long as the intervener does not take over the bulk of the fighting from one of the initial parties" (p. 4).
    \item Started prepping to merge my datasets into one comprehensive one with all the data I need to get going with the quantitative analysis. This primarily requires categorizing each of these into categories based on whether they adequately measure what I'm looking for. Will have a full merge by early next week so I can get started with the QCA.
    \item Researched QCA software -- I have a few options, which are a Python package (which I did a little looking into but seems individually-maintained, which is not ideal), an R package, which has extensive documentation behind the history and design and comparison to other softwares, and one with a GUI run on who-knows-what. While the latter two seem very usable, I'll probably stick with the R package for Linux compatibility (though try out the one with the GUI -- I find that most social science people prefer those kinds of softwares and it would be nice to be familiar with it).
\end{enumerate}
