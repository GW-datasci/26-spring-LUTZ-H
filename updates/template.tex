\section*{Challenges (3 points)}
% List below at least THREE significant challenges
\begin{enumerate}
	\item Unfortunately, in trying to get things to work for the QCA, I had to eliminate some data I planned to use. It would have restricted my dataset a little to include so much public opinion data, but I was willing to live with that, but upon implementing it I learned that the questions were varying too much between years for to use them without extensively consulting every survey wave's codebook, which would take a \emph{while} and be entirely manual. I can't justify that right now, since I need to work on outputting something for certain, but it's something to work on if I have time.
	\item The calibration/deciding between crisp and fuzzy sets is something that is somewhat difficult for the time being, as I am having to set the parameters that will influence my conclusions, and I worry that how that calibration is done is impacting my preliminary results too much. I'm not sure how to conduct a sensitivity analysis for this sort of thing, so it'll just be more reading.
	\item Debugging code problems with the QCA package is a pain, as some of the error handling isn't as specific or focused on what I'm doing as I would like, and frequently I would get errors that would involve the way that the package was actually programmed, which I am not familiar with. Everything was something I could handle, fortunately, but it's a pain.
\end{enumerate}
